# TO_DO_LIST_APP
Django app for a to do list
In the command line terminal, navigate to the directory where you saved this GitHub repository. Run the below commands to run the django app:
1. pip install django
2. python manage.py migrate
3. python mange.py runserver
4. Copy the url (http://127.0.0.1:8000/) into your browser and you should then be able to view the app.
